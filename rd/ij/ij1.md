# IntelliJ on Mac Shortcuts
### Mac keyboard symbols and modifiers description
Symbol|Description|说明
---|---|---
⌘|Command|命令  
⇧|Shift|换挡
⌥|Option|选项
⌃|Control|控制
↩︎|Return/Enter|回车
⌫|Delete|删除前面字符
⌦|Fn + Delete|删除后面字符
↑|Up|上箭头
↓|Down|下箭头
←|Left|左箭头
→|Right|右箭头
⇞|Page Up(Fn + ↑)|向上翻页
⇟|Page Down(Fn + ↓)|向下翻页
⇥|Tab|右制表符
⇤|Shift + Tab|左制表符
⎋|Escape(Esc)|退出
End|Fn + →|结束位置
Home|Fn + ←|开始位置
